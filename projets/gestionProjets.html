Trelltech

Application web (ou desktop/mobile) de gestion de projet, développée avec l’API Trello pour simplifier la création et le suivi de workspaces, tableaux, listes et cartes. L’accent est mis sur la fluidité UX/UI, la robustesse de l’architecture et la qualité du code.

Fonctionnalités principales

  • Espaces de travail : création / mise à jour / suppression / affichage.
  • Tableaux : choix du template (Kanban…), création / modification / suppression / affichage.
  • Listes : gestion complète (CRUD).
  • Cartes : création, édition, suppression, affichage au sein d’une liste.
  • Attribution : assignation de membres à chaque carte.

Conception & Technologies

  • Intégration de l’API Trello (authentification, endpoints, Webhooks).
  • Framework choisi : [React / Vue / Flutter] (à compléter selon ton implémentation).
  • Tests : stratégie de tests unitaires et d’intégration avec [Jest / Mocha / JUnit].
  • Documentation : README détaillé, guide d’installation et diagrammes UML.

UX & UI

Création d’une identité visuelle cohérente (couleurs, icônes, typographie), respect des guidelines Material Design (ou équivalent), navigation ergonomique et retour utilisateur en temps réel.
