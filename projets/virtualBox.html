Création de 4 Machines Virtuelles

Ce projet comprend la création et l’automatisation de 4 machines virtuelles ayant chacune un rôle distinct : passerelle, serveur web, poste employé et poste administrateur.

Détail des Machines

1. Passerelle (OpenBSD 7.6)

La VM « Passerelle » sert à relier et isoler plusieurs réseaux. Système : OpenBSD 7.6.

  • 4 cartes réseau :
  • • NAT : traduction d’adresses réseau
  • • 3 réseaux privés : attribution d’identités distinctes à chaque VM
  • • Adaptateur réseau virtuel actif pour la connexion

2. Serveur Web (FreeBSD 14)

La VM « Serveur Web » héberge une application PHP et expose une base de données. Système : FreeBSD 14.

  • Réseau interne connecté au serveur DHCP de l’administration
  • Base de données MySQL/PostgreSQL accessible via script PHP

3. Client (Employé)

Simule le poste d’un employé standard. Système : Debian 12 (ou autre distrib légère).

  • Réseau privé isolé
  • Interface graphique (XFCE/GNOME) pour utilisation bureautique

4. Client (Administrateur)

Simule le poste d’un administrateur réseau. Système : Debian 12 (ou distrib adaptée).

  • Réseau privé avec accès privilégié
  • Outils d’administration (SSH, monitoring, gestion de VM)
